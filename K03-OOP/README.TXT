------------------------------------------------------------------------
OOP = Objekt-orientierte Programmierung
------------------------------------------------------------------------

* Ein Objekt ist eine Sammlung von Daten und Funktionen, 
  die zusammen gehören
* Hilft dabei, die Wirklichkeit in Simulation abzubilden
* Verbessert die Lesbarkeit von Programme, da sie sinnvoll
  gegliedert werden können <=> Spaghetti-Code
  

* Instanzvariablen
    * sollen immer "private" sein
    * mit "final" bezeichnet man Konstanten, 
      die ihren Wert nicht ändern
